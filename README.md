# AI Spend Audit SaaS

A production-ready SaaS application that audits a company's AI tool spending and recommends cost-saving alternatives.

## Overview

This product analyzes current spend on AI tools (such as Cursor, GitHub Copilot, ChatGPT, Claude, Anthropic API, etc.) and uses a deterministic, rule-based audit engine to provide actionable recommendations for savings, better alternatives, or optimized plans.

## Tech Stack

**Frontend:**
- React 19 + Vite
- TypeScript
- TailwindCSS
- shadcn/ui
- React Router & TanStack Query

**Backend:**
- Node.js + Express
- TypeScript
- MongoDB Atlas (Mongoose)

**Infrastructure:**
- Vercel (Frontend Deployment)
- Render (Backend Deployment)
